# Security Policy

## Reporting a Vulnerability

If you find a security issue in this repository, do not open a public issue with exploit details.

Report it privately through the repository hosting platform's security advisory flow if available. If that is not available, contact the maintainer directly and include:

- A clear description of the issue
- Steps to reproduce it
- The affected files or commands
- Any mitigation you have already verified

You can expect an acknowledgment within 5 business days.

## Scope

This repository is a local scanner and documentation skill. The most likely security issues are:

- Unsafe file handling or path traversal in the scanner
- Packaging or release issues that ship broken or unexpected artifacts
- Documentation that encourages unsafe repository practices

## Disclosure

Please allow time for a fix before public disclosure.
